\section{Defining Zero Sum Game Problem}\label{sec:defining-zero-sum-game-problem}
\begin{itemize}
	\item Defining a set of possible states: State Space
	\item Specify how will you track Whose Move / Turn it is
	\item Specify Initial State
	\item Specify Goal State(s) (there can by multiple)
	\item Define a FINITE set of possible Actions (legal moves) for EACH state in the State Space
	\item Come up with a Transition Model which describes what each action does
	\item Come up with a Terminal Test that verifies if the game is over
	\item Specify the Utility (Payoff/Object) Function: a function that defines the final numerical value to player $p$ when the game ends in the terminal state $s4$
\end{itemize}

\section{Constraint Satisfaction Problem}\label{sec:constraint-satisfaction-problem}
The goal is to find an assignment (variable = value):
\[ X_{1} = v_{1}, \dots, X_{n} = v_{n} \]
\begin{itemize}
	\item If NO constraints violated: consistent assignment
	\item If ALL variables have a value: complete assignment
	\item If SOME variables have NO value: partial assignment
	\item SOLUTION: consistent and complete assignment
	\item PARTIAL SOLUTION: consistent and partial assignment
\end{itemize}
